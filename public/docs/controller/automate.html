Audio Controller

Automates the output-connected audioparam based on the specified duration and envelope.

Inputs

trigger (event)
Starts the automation of connected audioparam upon receiving an event on this terminal.

Outputs

out (audio)
The audio output that can be connected to any audioparam for its automation.

Properties

Min The minimum threshold for audioparam
Max The maximum threshold for audioparam
Duration The duration of automation
Auto Start Whether to enable auto-play
Loop Whether to loop automation infinitely

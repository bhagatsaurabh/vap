Audio Source

Creates an audio source from a file.

Inputs

array-buffer (event)
Configures a new audio source based on an event received on this terminal.
gain (audioparam)
Controls gain of audio output.
detune (audioparam)
Controls tune adjustment of audio output.
playback-rate (audioparam)
Controls output playback rate.

Outputs

out (audio)
The audio output.
ended (event)
Fires an event on this terminal when audio playback has finished.

Properties

File The audio source file
Loop Whether to loop upon audio playback completion

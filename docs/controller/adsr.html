Audio Controller

Attack, Decay, Sustain & Release envelope, can be used to vary audioparams over time.

Inputs

trigger (event)
Toggles between attack and release upon receiving an event on this terminal.

Outputs

out (audio)
The audio output that can also be used as a controlling signal for audioparams.

Properties

Min The minimum value this envelope can output
Max The maximum value this envelope can output
A Duration for attack
D Duration for decay
S Duration for sustain
R Duration for release

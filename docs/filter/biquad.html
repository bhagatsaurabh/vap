Audio Controller

Applies frequency shaping on input audio.

Inputs

in (audio)
The input audio.
gain (audioparam)
The gain control.
Q (audioparam)
The quality factor control.
frequency (audioparam)
The frequency control.
detune (audioparam)
The detune control.

Outputs

out (audio)
The filtered audio output.

Properties

Type The type of filter
Bypass Whether to bypass input audio processing
